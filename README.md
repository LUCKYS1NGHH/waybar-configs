## Setup 
```bash
git clone https://github.com/LUCKYS1NGHH/waybar-config
cd waybar-config
mkdir -p ~/.config/waybar
```

> Choose the style you liked in screenshots and copy-paste that directory,
> example:

```bash
cp -r 1/ ~/.config/waybar
```
