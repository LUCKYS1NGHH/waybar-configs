## Setup 
```bash
git clone https://github.com/LUCKYS1NGHH/waybar-config
cd waybar-config
mkdir -p ~/.config/waybar
```

> Choose the style you liked in screenshots and copy-paste that directory,
> Example:

```bash
cp -r 1/* ~/.config/waybar/
```

> [!NOTE]
There are few configs like 3 which have scripts need to configured little bit, example for weather city and wallpaper change path
